# Betabots v0.1.0

Betabots v0.1.0 is the first stable release candidate for local open-source installation across Codex, Agent Skills, Claude Code, and Cursor.

## Highlights

- Pinned Playwright runtime dependency for the repository and copied skill installs.
- Clean install path that does not rely on sibling repositories, global npm packages, or `NODE_PATH`.
- Installer now places runtime dependencies in each copied Betabots skill directory.
- Explicit Chromium-only browser install command.
- Optional Linux system dependency installation for minimal images.
- Automated clean-install verifier for release checks.

## Install

```bash
git clone https://github.com/yevgeniusr/betabots.git
cd betabots
node scripts/install-deps.cjs --all
node scripts/install-browsers.cjs
tests/smoke.sh
scripts/install-local.sh all
```

`node scripts/install-browsers.cjs` downloads Chrome for Testing, Chrome Headless Shell, and FFmpeg for Chromium support for the pinned Playwright revision. It does not download Firefox or WebKit.

On minimal Linux images, run `node scripts/install-browsers.cjs --with-deps` if Playwright reports missing system libraries. That path may invoke the platform package manager.

## Verify

```bash
node scripts/verify-clean-install.cjs --skip-browser-install
```

Use `--skip-browser-install` only when the CI image or machine has already run the Chromium install step for the pinned Playwright revision. With this flag, the verifier reuses the caller's Playwright browser cache, or the cache named by `PLAYWRIGHT_BROWSERS_PATH`, while keeping the temporary repository and dependency tree isolated. Omit it to let the verifier run the browser install command.
