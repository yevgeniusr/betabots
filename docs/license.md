# AGPLv3 License Guide

Betabots uses the [GNU Affero General Public License v3.0 or later](../LICENSE).

This is a plain-English guide, not a replacement for the license text.

## You Can

- Use Betabots for personal, hobby, student, research, internal, and commercial projects.
- Copy, fork, and study the project.
- Modify it.
- Share original or modified versions when you follow the license.

## You Must

- Keep copyright notices and the license.
- License distributed modified versions under AGPLv3-compatible terms.
- Provide corresponding source code when you distribute Betabots or modified versions.
- Provide corresponding source code to users who interact with a modified version over a network.

## Why AGPL

AGPLv3 is designed for software that may be run as a network service. It keeps improvements public even when a modified version is offered through a web app, API, hosted agent service, or other network interface.

## Commercial Use

Commercial use is allowed, but commercial users must follow the same AGPLv3 obligations as everyone else.

## Practical Note

If you only use an unmodified copy privately, AGPLv3 has minimal practical friction. If you modify Betabots and share it, distribute it, or make it available to users over a network, publish the corresponding source under the same license.
